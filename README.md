# Deeplenz Frontend 

A modern, multilingual corporate website for Deeplenz — an AI & technology solutions company based in Saudi Arabia.

##  Live Demo
https://deeplenze-oc5r.vercel.app

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- English & Arabic (RTL/LTR) Support

## Getting Started

### Install
```bash
npm install
```

### Run
```bash
npm run dev
```

### Build
```bash
npm run build
```
